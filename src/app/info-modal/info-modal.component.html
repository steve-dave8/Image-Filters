Image Filters App

Usage

Summary: upload an image, apply filters, and download the image with filters applied.

  1. Upload an image file using one of these methods:
    • click the upload button at the top (Desktop, Mobile)
    • drag and drop an image onto the designated drop area (Desktop)
  2. Using the toolbar on the left, apply filters to the uploaded image.
  3. Click the download button at the bottom to download the filtered image (png file type).
  4. Repeat to generate more filtered images.
    • With the same image uploaded, change the active filters and download a different version of the image.
    • Upload another image. The filters selected will automatically be applied upon upload. This is useful if you want to apply the same filters to multiple images.

Notes

  • Filters can interact with each other in interesting ways. Experiment with mixing multiple filters for different effects.
  • Some filters do not work in all browsers.
